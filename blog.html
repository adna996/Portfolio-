Data Structures

Title description, Dec 1, 2019

A data structure is a particular way of organizing data in a computer so that it can be used effectively.To get you started, let’s first take a deep dive into Search and Sort, two classes of algorithms you can’t live without.

  • Searching
  • Sorting

Searching

Roughly speaking, there are two categories of search algorithms you’ll need to know right away: linear and binary. The linear and binary algorithms are named as such to describe how long (time complexity) a search is going to take based on the size of the input that is being search.In short, linear = simple.So let's complicate things. What is binary search? Search a sorted array by repeatedly dividing the search interval in half. Begin with an interval covering the whole array. If the value of the search key is less than the item in the middle of the interval, narrow the interval to the lower half. Otherwise narrow it to the upper half. Repeatedly check until the value is found or the interval is empty.

Sorting

Sorting Algorithms Best Time Complexity Average Time Complexity Worst Time Complexity Space Complexity Stable?
QuickSort O(logN) O(NlogN) O(N²) logN or N NO
MergeSort O(NlogN) O(NlogN) O(NlogN) O(N) YES
InsertionSort O(N) O(N²) O(N²) O(1) YES
HeapSort O(NlogN) O(NlogN) O(NlogN) O(1) NO

When choosing a sorting algorithm to use, weigh these factors. For example, quicksort is a very fast algorithm but can be pretty tricky to implement; bubble sort is a slow algorithm but is very easy to implement. To sort small sets of data, bubble sort may be a better option since it can be implemented quickly, but for larger datasets, the speedup from quicksort might be worth the trouble implementing the algorithm.

DATA STRUCTURES


A data structure is a way of organizing the data so that the data can be used efficiently. We have to know the difference between linear and non-linear data structurres. A data strucutre is said to be linear if its elements form a sequence or a linear lisr(array, linked list,etc). Non-linear structure is said to be if traversal of nodes is nonlinear in nature(grap, trees, etc).
I have to note that an array is the most fundamental data structure, which stores elements at a contiguous memory location. A linked list is another common data structure that complements the array structure. Similar to the array, it is also a linear data structure adn stores elements in a linear fashion.
